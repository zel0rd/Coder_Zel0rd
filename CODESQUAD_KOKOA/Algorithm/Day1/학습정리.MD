# JS에서 입력받기

## 0. 들어가기 전에

### - Interactive vs Non-Interactive Shell

쉘이란?
리눅스의 쉘은 명령어와 프로그램을 실행할 때 사용하는 인터페이스

쉘은 크게 두가지로 나눠짐

- Interactive shell
사용자로부터 프롬프트를 통해 직접 명령을 입력받아 실행
- Non-Interactive shell
작성한 Script파일을 통해서 실행

코딩테스트에서 입력을 받을 때, 쉘에 대해 알아야하는 이유는 바로 EOF가 다르기 때문이다.

콘솔에서 입력을 받게 될 때는 interactive shell이 계속 열려있기 때문에 자동으로 close가 호출되지 않는다.

하지만 일반적으로 알고리즘 코딩 사이트에서는 테스트 데이터를 파일에 저장한다. 그래서 콘솔에서의 키보드 입력과 달리, 파일 데이터 끝이 끝나면 자동으로 EOF에 걸려 Close가 된다.

그래서 콘솔에서 입력을 중단하고 싶을 때는 Ctrl + D라고 입력해야 한다.

## 1. readline 모듈사용

### 1.1 한 줄로 입력을 받을 때

```jsx
const readline = require("readline");
const rl = readline.createInterface({
	input: process.stdin,
	output: process.stdout
});

rl.on("line", function(line) {
	console.log("input : ", line);
	rl.close();
}).on("close", function() {
	console.log("readline close() called")
	process.exit();
});
```

### 1.2 여러 줄을 입력으로 받을 때(Ctrl + d)

```jsx
const readline = require('readline');
const rl = readline.createInterface({
	 input: process.stdin,
	 output: process.stdout
});

let input = [];

rl.on('line', function (line) {
		input.push(line)
	})
	.on('close', function () {
		console.log(input);
    process.exit();
});
```

## 2. fs 모듈사용 (Ctrl + d)

```jsx
var fs = require('fs');
var input = fs.readFileSync('/dev/stdin').toString().split(' ');

console.log(input)
```

## 3. 활용 예시

### - 두 수를 입력받아 합을 구하는 경우

```jsx
const readline = require('readline');
const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

let input = [];

rl.on('line', function (line) {
    input = line.split(' ').map((el) => parseInt(el));
    console.log(input)
    rl.close()
  })
  .on('close', function () {
    console.log(input[0] + input[1]);
    process.exit();
  });
```

### - 몇 개의 숫자를 입력 받을지 쓰고 입력을 받는 경우

```jsx
let fs = require('fs');
let input = fs.readFileSync('/dev/stdin').toString().split('\n');

let count = parseInt(input[0]);
let numbers = new Array(count).fill(0);

for (let i = 0; i < count; i++) {
  numbers[i] = input[i+1]
}

// 3
// 123
// 123
// 123
// Ctrl + d
```

## 4. 기타

- 코딩테스트에서 readline모듈로 시간초과가 나올 경우 fs모듈을 사용하면 개선될 수도 있음

# JS에서 range array 만들기

# 1. ...Array 사용하기 (spread operator)

```jsx
[...Array(5).keys()]
// [0,1,2,3,4]
```

# 2. Array.from() 사용하기

```jsx
Array.from({length: 5}, (x,i) => i)
// [0,1,2,3,4]
```

```jsx
Array.from(Array(5), (val,idx) => idx)
// [0,1,2,3,4]

// val => [ undefined, undefined, undefined, undefined, undefined]
// idx => [0,1,2,3,4]
```

# 3. Array.fill().map() 사용하기

```jsx
Array(5).fill(1).map((value,index) => value+index)
// [1,2,3,4,5]

Array(5).fill(0).map((value,index) => value+index)
// [0,1,2,3,4]

// value [0,0,0,0,0]
// index [0,1,2,3,4]
```

# 4. 활용하기

### - 특정 범위의 숫자로 배열 만들기

```jsx
function numberRange(start,end){
    return new Array(end - start).fill().map((_,i) => i + start)
}

numberRange(3,7)
// [3,4,5,6]
```

### - 특정 범위의 숫자를 일정한 간격으로 배열만들기

```jsx
function numberRange(start, end, step){
    return new Array( parseInt((end - start - 1) / step ) + 1).fill().map((_,i) => start + i*step)
}

console.log(numberRange(3,13,3))
// [ 3, 6, 9, 12 ]
```

# JS에서 정렬하기
```jsx
// 오름차순 정렬
function sortAsc(arr){
  arr = arr.sort(function(a,b){
    return a - b;
  })
  return arr
}
```

```jsx
// 내림차순 정렬
function sortDes(arr){
  arr = arr.sort(function(a,b){
    return b - a;
  })
  return arr
}
```

```jsx
arr = [3,2,4,2,5,6,3,4]

sortAsc(arr)
//[ 2, 2, 3, 3, 4, 4, 5, 6 ]

sortDes(arr)
// [ 6, 5, 4, 4, 3, 3, 2, 2 ]

```
